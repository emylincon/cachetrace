Emeka Ugwuanyi

Researcher, Lecturer and Developer

This application has been developed by Emeka Ugwuanyi, In contribution to his work on Caching in Multi-Access Edge Computing. The purpose of the application is to provide an interface that can be used to compare already existing caching algorithms.


This will primarily help researchers by saving time used on implementation of existing algorithms. Therefore, researchers can focus on design and development of their proposed algorithms and then use this interface to verify the benefits of the proposed scheme.


The caching algorithms available for comparison in the application include Least Frequently Used (LFU), Least Recently Used (LRU), First In First Out (FIFO), Least Frequently Recently Used (LFRU), Multi Queue (MQ), Frequency Based Replacement(FBR) and Optimal Page Replacement (OPR).


To use the application, specify the parameters as wanted. The comparison is made across cache sizes specified in an array format. A downloadable json file is available after each run.


API Support is currently under development and would be made available as soon as possible.
